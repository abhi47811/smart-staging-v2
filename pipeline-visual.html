Phase 7 — Refinement Layer
✂️
Edit Engine (M14) /api/edit + /api/edit/submit
Surgical regeneration. User picks an element → system computes influence zone (element + nearby affected elements) → inpaints only that region → creates new scene graph version (full undo/redo history). Actions: swap, move, resize, material_change, color_change, style_change, remove, add.
